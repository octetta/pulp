Skred

A minimalist sound engine, pattern engine, and live coding environment for wavetable and sample-based audio synthesis.

About the Demo

  • Multi-Track Sequencing: Features a drum pattern layout where each individual patch resides on a separate track, running independent delay configurations.
  • Sample Asset Handling: Track 3 demonstrates sample playback by loading an alto flute WAV file and respecting its embedded loop tags.
  • Architecture & Design: You can read about the engine's core goals, implementation criteria, and technical comparisons directly in the PULP Documentation Repository.
